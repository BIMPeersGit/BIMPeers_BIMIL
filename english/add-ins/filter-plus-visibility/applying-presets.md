# Applying Presets

## Toggle from Main Panel

<div><figure><img src="../../.gitbook/assets/image (101).png" alt=""><figcaption></figcaption></figure> <figure><img src="../../.gitbook/assets/image (103).png" alt=""><figcaption></figcaption></figure></div>

Click the toggle button on the right side of each preset to turn it on or off. Multiple presets can be active at the same time.

## Assign to Ribbon Buttons

<figure><img src="../../.gitbook/assets/image (107).png" alt="" width="270"><figcaption></figcaption></figure>

For quick access, assign presets to Ribbon button slots (#1–#12):

1. Click the bookmark column dropdown for the preset
2. Select a slot number

Once assigned, the preset can be toggled directly from the Revit Ribbon. The button icon reflects the current active state.

## Keyboard Shortcuts

Ribbon buttons can be assigned keyboard shortcuts through Revit's Keyboard Shortcuts settings, allowing even faster access without opening the main panel.
