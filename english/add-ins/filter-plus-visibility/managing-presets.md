# Managing Presets

## Main Panel

<figure><img src="../../.gitbook/assets/image (106).png" alt="" width="270"><figcaption></figcaption></figure>

The main panel displays all saved presets in a list.

### **Columns:**

* **Bookmark** – Assign a preset to a Ribbon button slot (#1–#12) for quick access
* **Preset** – Shows the mode (HIDE/ISO) and preset name, with category count in parentheses
* **Edit** – Opens the preset editor
* **Toggle** – Turn the preset on or off

### **Toolbar:**

* **Search** – Filter presets by name
* **+ New Preset** – Create a new preset

### **Sorting:**

Click on column headers to sort the list by bookmark slot, preset name, or active state.



## New Preset / Edit Preset

<figure><img src="../../.gitbook/assets/image (100).png" alt="" width="300"><figcaption></figcaption></figure>

When creating or editing a preset, the following options are available:

* **Name** – Enter a name for the preset
* **Type** – Choose between Hide (hide selected categories) or Isolate (show only selected categories)
* **Categories** – Select categories to include in the preset
  * Use the search bar to filter categories by name
  * Check "Show selected categories only" to view currently selected categories

Click **Save** to apply changes or **Cancel** to discard.



### Edit Preset:

The edit window provides the same options as creating a new preset, with one addition:

* **Delete Preset** – Permanently remove the preset
